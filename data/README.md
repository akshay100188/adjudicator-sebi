# data

- `raw/` — fetched source PDFs/HTML (gitignored; never redistributed as a corpus). Phase 2.
- `uploads/` — ephemeral document uploads (gitignored; never persisted long-term). Phase 1b.

Only **public** SEBI text. Non-commercial. Not redistributed as a standalone database (see
`docs/checklist.md`).
